Votre objectif est de mettre à jour toutes les dépendances vulnérables.

Procédez comme suit :

1. Exécutez `npm audit` pour trouver les packages installés vulnérables dans ce projet.

2. Exécutez `npm audit fix` pour appliquer les mises à jour.

3. Exécutez les tests et vérifiez que les mises à jour n'ont rien cassé
