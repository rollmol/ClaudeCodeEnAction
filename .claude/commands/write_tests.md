Conventions de test :

Utiliser Vitests avec la bibliothèque de test React

Placer les fichiers de test dans un répertoire **tests** du même dossier que le fichier source

Nommer les fichiers de test sous la forme [nom_de_fichier].test.ts(x)

Utiliser le préfixe @/ pour les importations

Couverture :

Tester les cas nominaux

Tester les cas limites

Tester les états d'erreur
